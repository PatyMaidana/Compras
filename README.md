# Aplicação de Sistema de Cartão de Crédito

## Descrição

O sistema permite o lançamento do valor de limite de compras.
Checa o limite a cada compra e informa caso saldo insuficiente.
Cada compra é lançada com seu descritivo e valor correspondente.
O sistema trabalha com loop while para o usuário escolher parar ou continuar as compras. 
Ao interromper o loop o sistema apresenta a lista, ordenada por valor, das compras efetuadas e o saldo restante.
